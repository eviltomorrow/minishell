# File Browser Tree View — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `t` toggle in the SFTP file browser that shows a 2-level deep indented tree view of the current directory, with navigation working and transfer/delete/rename disabled in tree mode.

**Architecture:** All changes are scoped to `minishell-tui/filebrowser.rs`. A new `TreeEntry { entry, depth }` struct is added alongside `PanelState.tree_mode` and `PanelState.tree_entries`. Two new functions build the flattened tree from filesystem/SFTP. The render path and key handler are updated to branch on `tree_mode`.

**Tech Stack:** ratatui, crossterm, std::fs, ssh2 sftp

## Global Constraints

- No changes outside `crates/minishell-tui/filebrowser.rs`
- No changes to `FileEntry` in `minishell-ssh/sftp.rs`
- Tree depth hard-coded at 2 (not configurable)
- Indentation only (no tree connector characters)
- File entries sorted: directories first, then alphabetical, at every depth level

```rust
fn build_tree_local(&self, path: &Path, max_depth: usize) -> Vec<TreeEntry> {
    self.build_tree_local_recursive(path, max_depth, 0)
}

fn build_tree_local_recursive(&self, path: &Path, max_depth: usize, depth: usize) -> Vec<TreeEntry> {
    let mut result = Vec::new();
    let mut errors = Vec::new();

    let read_dir = match std::fs::read_dir(path) {
        Ok(d) => d,
        Err(e) => {
            errors.push(format!("readdir {}: {}", path.display(), e));
            return result;
        }
    };

    let mut entries: Vec<FileEntry> = Vec::new();
    for entry in read_dir.flatten() {
        let name = entry.file_name().to_string_lossy().to_string();
        if name.starts_with('.') {
            continue;
        }
        let meta = entry.metadata().ok();
        let modified = meta
            .as_ref()
            .and_then(|m| m.modified().ok())
            .and_then(|t| t.duration_since(std::time::UNIX_EPOCH).ok())
            .map(|d| format_modified(Some(d.as_secs())))
            .unwrap_or_default();
        let perm = meta.as_ref().map(|m| {
            #[cfg(unix)]
            {
                use std::os::unix::fs::PermissionsExt;
                format_perm(Some(m.permissions().mode() & 0o777), m.is_dir())
            }
            #[cfg(not(unix))]
            { String::new() }
        }).unwrap_or_default();
        entries.push(FileEntry {
            name,
            is_dir: meta.as_ref().map(|m| m.is_dir()).unwrap_or(false),
            size: meta.as_ref().map(|m| m.len()).unwrap_or(0),
            modified,
            perm,
        });
    }
    entries.sort_by(|a, b| b.is_dir.cmp(&a.is_dir).then(a.name.cmp(&b.name)));

    for entry in entries {
        result.push(TreeEntry { entry: entry.clone(), depth });
        if entry.is_dir && depth < max_depth {
            let child_path = path.join(&entry.name);
            result.append(&mut self.build_tree_local_recursive(&child_path, max_depth, depth + 1));
        }
    }
    result
}
```

fn build_tree_remote(&self, sftp: &Sftp, path: &str, max_depth: usize) -> Vec<TreeEntry> {
    self.build_tree_remote_recursive(sftp, path, max_depth, 0)
}

fn build_tree_remote_recursive(&self, sftp: &Sftp, path: &str, max_depth: usize, depth: usize) -> Vec<TreeEntry> {
    let mut result = Vec::new();
    let mut errors = Vec::new();

    let entries = match sftp::list_dir(sftp, path) {
        Ok(e) => e,
        Err(e) => {
            errors.push(format!("listdir {}: {}", path, e));
            return result;
        }
    };

    for entry in entries {
        result.push(TreeEntry { entry: entry.clone(), depth });
        if entry.is_dir && depth < max_depth {
            let child_path = format!("{}/{}", path.trim_end_matches('/'), entry.name);
            result.append(&mut self.build_tree_remote_recursive(sftp, &child_path, max_depth, depth + 1));
        }
    }
    result
}
```

fn toggle_tree(&mut self) {
    let is_tree = {
        let p = self.active_panel();
        p.tree_mode
    };

    let panel = self.active_panel_mut();
    if is_tree {
        panel.tree_mode = false;
        panel.tree_entries.clear();
        self.refresh_panel(self.active_side);
        self.status = format!("{} entries", panel.entries.len());
    } else {
        if self.active_side == Side::Remote && self.session.is_none() {
            self.status = "Not connected".to_string();
            return;
        }

        if self.active_side == Side::Remote {
            let session = match self.session.as_ref() {
                Some(s) => s,
                None => return,
            };
            let sftp = match session.sftp() {
                Ok(s) => s,
                Err(e) => {
                    self.status = format!("SFTP error: {}", e);
                    return;
                }
            };
            let path = panel.current_path.to_string_lossy().to_string();
            panel.tree_entries = self.build_tree_remote(&sftp, &path, 2);
        } else {
            let path = panel.current_path.clone();
            panel.tree_entries = self.build_tree_local(&path, 2);
        }

        panel.tree_mode = true;
        panel.cursor = 0;
        panel.scroll_offset = 0;
        self.status = format!("{} entries (tree)", panel.tree_entries.len());
    }
}
```
